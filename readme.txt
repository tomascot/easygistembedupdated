Easily embed GitHub Gists just by pasting the Gist URL into a post/page.

== Description ==

This is a very simple plugin that will allow you to automatically embed GitHub Gists into posts/page just by entering the URL of a Gist.

**Support:** [Captain Theme](http://captaintheme.com/)

== Installation ==

How to install this plugin?

1. Search for "Easy Gist Embed" in the Add New Plugin section of your site or download it and upload it manually.
1. Install and activate the plugin.
1. Enter a link to a [GitHub Gist](http://gist.github.com/) into a post/page/anything.
1. Magic!

== Frequently Asked Questions ==

= So how do I embed the Gist? =
Just enter the link to the gist (eg. https://gist.github.com/3732870) in a post or page and it'll magically embed!
